# E3news

E3news is an RSS aggregator to feed news about several video games sources.

# Prerequisites

In order to self hosted your E3 News app, you have to install:

- git
- npm

``` shell
$ git clone https://github.com/LeyluIAA/E3news.git
$ cd E3news
$ npm install

```

# Launch

Just run

``` shell
$ node index.js

```
